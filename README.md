# Sudoku Solver

A full-stack Sudoku Solver application built with **Angular** and **NestJS**.

## Features

- ✅ Solve Sudoku puzzles entered manually
- ✅ Validate Sudoku input
- ✅ Highlight solved cells
- 🚧 Generate random Sudoku puzzles
- 🚧 Play Sudoku game
- 🚧 Upload Sudoku image and solve using OCR
- 🚧 Save puzzle history using MongoDB

## Getting Started

### Clone the repository

```bash
git clone https://github.com/sharma-psn/sudoku-solver.git
```

### Frontend

```bash
cd frontend
npm install
ng serve
```

Runs at:

```
http://localhost:4200
```

### Backend

```bash
cd backend
npm install
npm run start:dev
```

Runs at:

```
http://localhost:3000
```
